# 2026/8/7 陸委會座談會協調網頁

這是一個可直接放在 GitHub Pages 的靜態網站，用於展示與更新座談會議程。

## 頁面

- `index.html`：網站首頁
- `agenda.html`：單頁會議籌辦頁，包含可編輯議程、分工狀態與 DOCX 下載

## GitHub Pages

將此資料夾推上 GitHub 後，可在 repository 的 `Settings > Pages` 選擇從 `main` 分支根目錄發布。

純 GitHub Pages 是靜態網頁，議程頁的編輯狀態會先存在填寫者自己的瀏覽器。若要永久修改公開網站預設內容，需更新檔案後重新推送到 GitHub。
