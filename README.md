# 2026/8/7 陸委會座談會協辦確認頁

這是一個可直接放在 GitHub Pages 的靜態網站，用於協助中山大學與東海大學確認座談會議程與東海端協辦事項。

## 頁面

- `index.html`：首頁，說明東海需要完成的兩件事
- `agenda.html`：協辦頁，包含「議程內容」與「東海確認」兩個分頁

## 功能

- 議程欄位可直接點選修改，並可拖拉調整順序。
- 議程可透過瀏覽器列印功能另存為一頁式 PDF。
- 東海確認欄位有動作就會自動儲存在使用者自己的瀏覽器。

## GitHub Pages

將此資料夾推上 GitHub 後，可在 repository 的 `Settings > Pages` 選擇從 `main` 分支根目錄發布。

純 GitHub Pages 是靜態網頁，表單狀態會先存在填寫者自己的瀏覽器。若要永久修改公開網站預設內容，需更新檔案後重新推送到 GitHub。
