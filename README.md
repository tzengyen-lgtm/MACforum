# 2026/8/7 陸委會座談會協調網頁

這是一個可直接放在 GitHub Pages 的靜態網站，用於展示座談會議程與分工確認事項。

## 頁面

- `index.html`：網站首頁
- `agenda.html`：乾淨版議程，可依照網頁最新內容下載 DOCX 議程
- `tasks.html`：東海與中山分工確認，可勾選、儲存於瀏覽器，並產生回報文字

## GitHub Pages

將此資料夾推上 GitHub 後，可在 repository 的 `Settings > Pages` 選擇從 `main` 分支根目錄發布。

純 GitHub Pages 是靜態網頁，分工頁的勾選狀態會先存在填寫者自己的瀏覽器。若要讓主辦端收到更新，請使用分工頁的「產生回報」功能，複製文字或開啟 GitHub Issue 回報。
